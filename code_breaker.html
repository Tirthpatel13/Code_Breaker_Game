Code Breaker Rules

The objective of the game is to guess the secret code in a limited number of attempts. Each code consists of colored balls placed in a specific order. After each guess, you will receive feedback:

  • Exact Match: The correct color in the correct position.
  • Partial Match: The correct color in the wrong position.
  • No Match: A color that is not in the secret code.

Use these clues to solve the code before running out of attempts. Good luck!
